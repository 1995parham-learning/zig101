# Zig 101

Learning [Zig](https://ziglang.org/) programming language — started during the US/Israel war on Iran, March 2026.

I began this journey on February 28, 2026, the day the attacks started.
Between the sirens and the silence, there is code.
From Tehran, under bombing, I choose to learn rather than to freeze.

Because even in war, the compilers still work, and the mind still needs something to build.
